## 1) Headings (H1, H2, H3)

Use headings to organize content.

1. Use **H1 (`#`)** for the main page title (usually once).
2. Use **H2 (`##`)** for major sections.
3. Use **H3 (`###`)** for sub-sections inside an H2 section.

### Write this

```markdown
# Admissions 2026
## Eligibility
### Required Documents
```

### You get this

# Admissions 2026
## Eligibility
### Required Documents

---

## 2) Bold and Italic Text

### Write this

```markdown
Please submit **all documents** before the deadline.
The fee is *non-refundable*.
```

### You get this

Please submit **all documents** before the deadline.  
The fee is *non-refundable*.

---

## 3) Bullet Lists (unordered)

### Write this

```markdown
- Original marksheet
- Transfer certificate
- Passport size photos
```

### You get this

- Original marksheet
- Transfer certificate
- Passport size photos

---

## 4) Numbered Lists (ordered)

### Write this

```markdown
1. Fill the form
2. Upload required documents
3. Pay the application fee
```

### You get this

1. Fill the form
2. Upload required documents
3. Pay the application fee

---

## 5) Links (clickable text)

### Write this

```markdown
[Download Admission Form](https://example.com/admission-form.pdf)
```

### You get this

[Download Admission Form](https://example.com/admission-form.pdf)

---

## 6) Images (supported in this CMS)

Use the image upload button in the CMS, then paste/use the image URL.

### Write this

```markdown
![Campus Building](https://example.com/images/campus.jpg)
```

### You get this

![Campus Building](https://example.com/images/campus.jpg)

Tip: Always write clear image text inside `[]` so others understand what the image is.

---

## 7) Line Breaks and Paragraphs

This is where most mistakes happen.

1. Leave a **blank line** between paragraphs.
2. For a line break in the same paragraph, end a line with **two spaces**.

### Write this

```markdown
This is paragraph one.

This is paragraph two.

Admissions office timing: 10 AM to 5 PM.  
Sunday closed.
```

### You get this

This is paragraph one.

This is paragraph two.

Admissions office timing: 10 AM to 5 PM.  
Sunday closed.

---

## 8) Blockquotes

Use for important notes, highlights, or quoted messages.

### Write this

```markdown
> Last date to apply: 30 June 2026.
```

### You get this

> Last date to apply: 30 June 2026.

---

## 9) What Not To Do

1. Do not use too many headings in a row.
2. Do not make everything bold.
3. Do not paste directly from Word without cleanup (extra spaces and odd formatting may appear).
4. Do not use all caps for full paragraphs.
5. Do not paste long raw URLs without link text.
6. Do not skip preview before saving.

Quick fix after pasting from Word:

1. Paste into plain text first (or clear formatting).
2. Re-add headings, bold, lists, and links in Markdown.

## One-Page Cheat Sheet

| Syntax | What it does | Example |
|---|---|---|
| `# Heading` | Main title (H1) | `# Admissions 2026` |
| `## Heading` | Section title (H2) | `## Eligibility` |
| `### Heading` | Sub-section (H3) | `### Required Documents` |
| `**text**` | Bold text | `**Important**` |
| `*text*` | Italic text | `*Note*` |
| `- item` | Bullet list | `- Photo ID` |
| `1. item` | Numbered list | `1. Fill form` |
| `[Text](URL)` | Clickable link | `[Download](https://example.com/file.pdf)` |
| `![Alt](URL)` | Image | `![Campus](https://example.com/campus.jpg)` |
| Blank line between text blocks | New paragraph | `Line 1` + blank line + `Line 2` |
| Two spaces at line end | Line break (same paragraph) | `Office timing: 10-5.␠␠` then next line |
| `> text` | Highlight/quote block | `> Last date: 30 June` |
